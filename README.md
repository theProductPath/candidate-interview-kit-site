# Candidate Interview Kit — site companion

The marketing and landing pages for [Candidate Interview Kit](../candidate-interview-kit/README.md). This folder holds the website source, assets, and deployment configuration for `interview-kit.theproductpath.com`.

For the product itself (concepts, spec, implementation), see [`../candidate-interview-kit/`](../candidate-interview-kit/README.md).

## GitHub Pages

This repo publishes the static site from the repository root with GitHub Pages. The deployment workflow is in `.github/workflows/pages.yml`, and the custom domain is preserved in `CNAME`.

The Pages source for this repository should be set to **GitHub Actions** in the repository settings.
